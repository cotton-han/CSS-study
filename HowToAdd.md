# How To Add CSS

### External CSS

- `<head>` 태그 안에 `<link rel="stylesheet" href-"index.css">`

### Internal CSS

- `<head>` 태그 안에 바로 `<style>` 안에서 스타일 코드 작성

### Inline CSS

- `<h1 style="color:blue;">test</h2>`

> External 과 Internal 에서 동일한 요소에 스타일을 지정할 경우 마지막에 지정한 스타일이 적용
>
> Inline style은 stylesheet의 장점을 잃게 하기 때문에 사용하지 않는 것을 권장
